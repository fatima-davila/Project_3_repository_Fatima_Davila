SUMMARY: In this project we analyzed data from  National Household Travel Survey (NHTS) and the Next Generation Simulation (NGSIM). The link to those csvs can be found in the files. Our objective was to create a box plot, histogram, timeseries plot, bar plot, and a simulation using real and simulated data. The purpose of this is so that the client has a better understanding of the transportation data and what it is saying by plotting different variables. This repository holds the Gantt chart which has the tasks that were done to complete the objective. It has the timesheet that records all the time spent on this project. Project 3 scope of work describes the goals and tasks and deliverables in better detail. the annotated code document give an explanation on each line of code used in this project. "project 3 final code davila" is the .ipynb which holds the code for this project. The report is also in this repository and in there is the results of this project.
